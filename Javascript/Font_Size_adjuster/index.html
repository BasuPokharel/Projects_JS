The default font size is : 16px

I have created a simple yet interactive project that allows users to dynamically change the font size of text using an input field and a button. The project accepts a numerical value (in pixels) from the user through the input field, and upon clicking the button, it applies the entered font size to the text. This project demonstrates practical JavaScript functionality by handling user input, manipulating the DOM, and updating CSS properties in real time. It provides an easy way to experiment with different font sizes, making it a useful tool for understanding dynamic styling with JavaScript.
